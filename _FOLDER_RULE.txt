FOLDER RULE (Repository Guidelines):
==================================================
ALL new scripts MUST go into the root workspace.

Naming convention:
    ##_Game_Name
    Example: 24_My_New_Game

New files added to existing folders are tracked here.

CONTRIBUTION RULES:
1. For any new game script, create a folder following the naming convention
2. The loader script should display a game selection list with custom names
3. User must select the game and click "Load" to run the game-specific script
4. You must ALWAYS ask for explicit permission BEFORE adding/registering a new game
5. If you make a script, you must make a UI for it and ensure all scripts work inside it
6. Before writing ANY script for a new game, perform full reconnaissance
   and save report to the MCP_Access folder
==================================================
